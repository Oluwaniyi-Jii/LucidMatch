# LucidMatch Audit Log Template

This document provides the template structure for audit logs generated by the LucidMatch system.

## Individual Match Audit Log

```
================================================================================
LUCIDMATCH DECISION AUDIT LOG
================================================================================

Audit ID: [UUID]
Generated: [ISO 8601 Timestamp]
API Version: 0.1.0

================================================================================
CANDIDATE SUMMARY (Anonymized)
================================================================================

Experience Level: [beginner/intermediate/advanced/expert]
Years of Experience: [X years]
Education Level: [Degree type or "Not specified"]
Primary Competency Clusters: [List of top 3 clusters]
Confidence in Profile Data: [X%]

Note: Personal identifying information has been stripped. This analysis is based 
solely on skills, experience, and qualifications.

================================================================================
JOB MATCH RESULT
================================================================================

Position: [Job Title]
Department: [Department]
Match Score: [XX%] (±[Y%] confidence range)
Competency Level: [STRONG / MID / LOW]

--------------------------------------------------------------------------------
SCORE BREAKDOWN (1-10 scale)
--------------------------------------------------------------------------------

  Skills Match:      [█████████░] 9/10
  Experience Match:  [███████░░░] 7/10
  Education Match:   [██████░░░░] 6/10
  Achievement Match: [████████░░] 8/10

--------------------------------------------------------------------------------
SKILL TRANSFER ANALYSIS
--------------------------------------------------------------------------------

The following skills were evaluated for transferability:

| Candidate Skill    | Job Requirement  | Transfer % | Reasoning                    |
|--------------------|------------------|------------|------------------------------|
| [Skill A]          | [Requirement 1]  | [XX%]      | [Brief explanation]          |
| [Skill B]          | [Requirement 2]  | [XX%]      | [Brief explanation]          |
| ...                | ...              | ...        | ...                          |

--------------------------------------------------------------------------------
SKILL GAPS IDENTIFIED
--------------------------------------------------------------------------------

Priority: CRITICAL
  • [Skill Name] - Current: [Level/None], Required: [Level]
    Estimated Learning Time: [X weeks]

Priority: IMPORTANT
  • [Skill Name] - Current: [Level/None], Required: [Level]
    Estimated Learning Time: [X weeks]

Priority: NICE TO HAVE
  • [Skill Name] - Current: [Level/None], Required: [Level]
    Estimated Learning Time: [X weeks]

================================================================================
REASONING SUMMARY
================================================================================

[Plain English paragraph explaining why this match score was assigned. This 
should be understandable by HR managers without technical background. Include:
- Key factors that influenced the score
- Why certain skills transferred well
- What concerns exist and why
- Overall assessment of candidate potential]

================================================================================
STRENGTHS & CONCERNS
================================================================================

STRENGTHS:
  ✓ [Strength 1]
  ✓ [Strength 2]
  ✓ [Strength 3]

CONCERNS:
  ⚠ [Concern 1]
  ⚠ [Concern 2]

================================================================================
BIAS AUDIT RESULTS
================================================================================

Fairness Score: [XX/100]
Audit Status: [PASSED / FLAGGED FOR REVIEW]

Bias Flags Detected: [N]

[If flags exist:]
--------------------------------------------------------------------------------
Flag #1: [SEVERITY: LOW/MEDIUM/HIGH]
Category: [socioeconomic_proxy / credential_bias / language_bias / other]
Indicator: [What triggered this flag]
Recommendation: [How to address or interpret this flag]
--------------------------------------------------------------------------------

[If no flags:]
No significant bias indicators detected in this evaluation.

================================================================================
COUNTERFACTUAL ANALYSIS
================================================================================

This analysis was conducted on anonymized data. The following information was 
excluded from the matching process:

  • Candidate name and contact information
  • Specific company names (replaced with industry descriptors)
  • University names (replaced with education level descriptors)
  • Demographic indicators
  • Geographic information (except as relevant to experience)

Statement: The match score would remain unchanged if candidate demographic 
characteristics were different, as these factors were not inputs to the matching 
algorithm.

================================================================================
UPSKILLING RECOMMENDATIONS
================================================================================

[If upskilling path was generated:]

QUICK WINS (1-2 weeks):
  1. [Course Title] - [Provider]
     Skill: [Skill addressed]
     Duration: [X hours]
     Link: [URL]

  2. [Course Title] - [Provider]
     ...

CORE LEARNING (1-2 months):
  1. [Course Title] - [Provider]
     Skill: [Skill addressed]
     Duration: [X hours]
     Link: [URL]

TOTAL ESTIMATED UPSKILLING TIME: [X weeks]

PRIORITY ORDER: [Skill 1] → [Skill 2] → [Skill 3]

================================================================================
DECISION METADATA
================================================================================

Processing Time: [XXXX ms]
AI Model Used: [Model name - version]
Privacy Mode: [Enabled/Disabled]
Bias Check: [Enabled/Disabled]

================================================================================
CERTIFICATION
================================================================================

This audit log was automatically generated by LucidMatch v0.1.0.
All decisions were made using AI-assisted matching with human-interpretable 
reasoning.

For questions about this decision, contact your HR administrator or submit 
feedback through the Governance Panel.

================================================================================
END OF AUDIT LOG
================================================================================
```

## Batch Processing Audit Summary

For bulk candidate processing, a summary report is generated:

```
================================================================================
LUCIDMATCH BATCH AUDIT SUMMARY
================================================================================

Batch ID: [UUID]
Processing Date: [Date]
Total Candidates: [N]
Total Jobs Matched Against: [M]

--------------------------------------------------------------------------------
AGGREGATE STATISTICS
--------------------------------------------------------------------------------

Average Match Score: [XX%]
Score Distribution:
  Strong (75%+):  [N] candidates ([X%])
  Mid (50-74%):   [N] candidates ([X%])
  Low (<50%):     [N] candidates ([X%])

Average Processing Time: [XXXX ms] per candidate

--------------------------------------------------------------------------------
FAIRNESS METRICS
--------------------------------------------------------------------------------

Candidates Passing Audit: [N] ([X%])
Candidates Flagged for Review: [N] ([X%])

Bias Flag Distribution:
  Socioeconomic Proxy: [N] flags
  Credential Bias: [N] flags
  Language Bias: [N] flags
  Other: [N] flags

No concerning patterns detected in aggregate analysis.
[OR]
⚠ ATTENTION: The following pattern requires review: [Description]

--------------------------------------------------------------------------------
RECOMMENDATIONS
--------------------------------------------------------------------------------

[Based on aggregate analysis, any recommendations for improving the matching 
process or addressing systemic issues]

================================================================================
END OF BATCH SUMMARY
================================================================================
```
